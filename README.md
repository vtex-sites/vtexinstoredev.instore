# VTEX inStore Dev

## Setup

```bash
yarn
```

(must have npm permission to use private @vtexlab packages)

## Run

```bash
yarn start
```

## Build

```bash
yarn build
```

To test build:

```bash
yarn serve
```

## Test

```bash
yarn test
```

## To launch another version

```bash
Open a PR to this repo (probably updating @vtexlab/gatsby-theme-instore-core version)
```
